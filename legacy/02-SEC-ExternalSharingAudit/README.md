# External Sharing Audit Script

This script scans SharePoint Online and OneDrive for Business to identify files and folders that have been shared with external users.

## Key Features
*   Audits all SharePoint sites and OneDrive accounts.
*   Lists all files and folders shared externally.
*   Provides details on who shared the resource and with whom it was shared.
*   Exports the findings to a CSV report for security and compliance reviews.
